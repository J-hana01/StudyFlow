# StudyFlow

StudyFlow는 CRC(Clinical Research Coordinator)를 위한 일정 및 연구 관리 플랫폼입니다.

## 주요 기능

- 일정 관리
- Subject Visit 관리
- Monitor Visit 관리
- IRB 일정 관리
- SAE 관리
- 알림 기능
- PC ↔ 모바일 동기화
- Google Calendar 연동
